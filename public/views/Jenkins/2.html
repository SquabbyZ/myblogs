    zhuhaifeng 2019-10-31 Jenkins

    # Jenkins 修改端口号（windows 系统）

    windows 下改端口号： Jenkins 安装成功后，默认的端口为 8080。 如果该端口号与其他服务的端口号冲突，则需要更改 Jenkins 的端口号。 具体方法为：

    修改 Jenkins 安装目录下的 jenkins.xml 文件，将<arguments>元素中的 httpPort 的值 8080 改为其他值并保存，如下图。(因为我的 gitlab 的端口是 8443 ,所以这里我使用的是 8444 端口给 Jenkins)
    1

    一定要记得重启服务才会生效

    # Jenkins 重启和关闭

    我们在配置好 Jenkins 之后, Jenkins 的服务会自动的加入到 windows 的任务管理器中的服务中。

    # Jenkins 简易重启和关闭

    1.右键点击最下方的带有 开始 的任务栏的空白处

    2.选择任务管理器后,会弹出下面的窗口,选择标红的服务

    3.在该服务处点击右键会弹出菜单,来选择是停止服务、关闭服务还是重启服务

    # Jenkins 命令重启和关闭

    如果你使用的不是 Windows 10 系统,我建议你安装一个 git for Window———— git.bash 命令窗口 按住 shift 键 同时点击鼠标右键,如图:

    (ps:画红线上面的那个就是 git for window 的 git bash)

    # Jenkins 启动

    net start jenkins
    1

    # Jenkins 停止

    net stop jenkins
    1
