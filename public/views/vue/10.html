    在 Vue + ElementUi 中使用 input 组件自动获取焦点

    zhuhaifeng 2020-04-25 Vue

    # 解决方法是在组件内部自定义一个指令 v-focus

      directives: {
    focus: {
      inserted: function(el) {
        el.querySelector("input").focus();
      }
    }
